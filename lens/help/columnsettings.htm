Dynamic Editing with Column Settings

From the phpLens grid, you can click the gray Edit Column Settings icons to configure column specific settings. Each field or column can be configured individually. In this example, we can see the settings for the QuantityPerUnit column.

Each screen can have a different label attached to a field. Here we see that the field name has been abbreviated to Qty/Unit.

Other settings that can be configured include powerLens, colorLens, data validation and charting (if the column is numeric).

Beside each option is online help that explains the property in more detail.
