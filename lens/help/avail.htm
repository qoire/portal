Property Availability by Product

Table of Contents

  • Free phpLens has unlimited viewing of data, but no powerLens, editing, charting nor searching.
  • Basic phpLens has powerLens, charting and searching, but no editing.
  • Professional phpLens allows creating and editing of records and javascript validation. Does not support enterprise databases (see below).
  • Advanced phpLens has template and stored procedure support, and can be used with open source databases such as MySQL and PostgreSQL, and file based databases such as MS Access and VFP.
  • Enterprise phpLens has the same functionality as Advanced and supports enterprise databases such as Oracle, Microsoft SQL Server, DB2, Sybase and Informix, and all databases supported by Advanced.
