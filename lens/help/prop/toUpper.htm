toUpper
Convert all strings in text boxes to uppercase when saving
Default: false
Useful for databases that require fields to be in uppercase for indexing purposes such as Oracle. Fields that are longer than 64 chars long are not converted to uppercase.

Text entered in search screens is also uppercased.

You can change this 64 character setting in phplens/adodb/adodb.inc.php, class ADORecordSet->blobSize property.
