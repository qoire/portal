Filtering and Searching
sqlFilter
Sets the sql statement when searching/filtering
When we are viewing a single record and we want to perform a search, we don't want to search on that single record of course. We want to search on all the records in a table. So we provide an alternative SQL statement for searches.

Similarly, sometimes we have readonly columns which are normally hidden when browsing, but you might want to expose them for searching.

Syntax
# we search on name, category1, category2 in table
$lens->sqlFilter = 'select name,category1,category2 from table';

# the grid results that will be displayed
lens->sql = 'select name, gender, title, country from table';
