textAreaHeader
Attributes for <textarea> tag when editing/creating records
Default: 'rows=3 cols=40'
Use this to control the size of the text area. To change sizes of individual text area's, use the templates feature.

In phpLens 1.2 we have extended this property so that different textarea's can have different sizes. If you use this extension, you will need to define textarea sizes for all textareas used.

Syntax
# Define for all textareas:
$lens->textAreaHeader = 'rows=3 cols=80';

# Suppose you have 2 textareas linked to columns A and B:
$lens->textAreaHeader= 'A^rows=2 cols=60;B^rows=10 cols=60';
