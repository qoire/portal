ID to identify this phpLens object
You might have multiple phpLens objects configured on different pages. Each copy of phpLens must be assigned a unique ID. This ID must be no longer than 10 characters, and must consist only of underscores, letters or numbers (_ A-Z and 0-9).

This is also part of the primary key into the phpLens table, and used in naming GET and POST variables.
