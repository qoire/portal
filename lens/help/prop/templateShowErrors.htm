templateShowErrors
Place error messages within template instead of showing them automatically
Default: false
By default, we always display error messages above the phpLens grid.

However if you have custom screens, then you might need to customize the position of the error messages, or decide to hide them. Set this property to true to hide automatic display of error messages and to place the error messages in a template variable.

The error messages are stored in the {ERRORMESSAGES} variable.
