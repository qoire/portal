templateMode
Controls how edit/new templates display input fields
This is a set of bit fields that control how input fields are displayed in edit and new record templates.

Bit field 1: Setting it means that the _FORMBEGIN_ and _FORMEND_ values will be automatically added to the beginning and end of the template.

Bit field 2: Setting it means that the field titles will be added above the input fields automatically, so you don't have to place {$COLUMNNAME_T} variables in the template yourself.

Syntax
$lens->templateMode = 3; // add _FORM*_ values and show field titles

$lens->templateMode = 2; // show field titles only

$lens->templateMode = 1; // add _FORM*_ values only
