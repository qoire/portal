imageAttributes
Attributes in the <img> tag such as WIDTH or HEIGHT
Default: ''
Default is to let the browser size the images. You can override this here.

Syntax
If you want all images to be 100 pixels wide and 120 pixels high: $lens->imageAttributes='WIDTH=100 HEIGHT=100';
