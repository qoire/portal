Description:
AWStats is a tool that analyzes your web server log files (IIS 5.0+, Apache or other web/proxy/wap server) to show you in HTML pages advanced web statistics including visitors, pages, hits, hours, search engines, keywords used to find your site, broken links, robots visits and more.You can run awstats from command line or from your browser as a CGI.Distributed under GNU General Public License.Supports multiple languages.
